
# Grocery Store Web Application

## Overview
This is a full-featured Grocery Store web application built using PHP and MySQL. The application allows users to browse products, add items to the cart, register, login, and place orders. An admin interface is included for managing products and orders.

This project demonstrates full-stack web development skills including server-side scripting, database integration, authentication, and dynamic page rendering.

---

## Features
- User registration and login
- Browse grocery products by categories
- Add products to cart and wishlist
- Place orders and view order history
- Admin panel for product and order management
- Search functionality
- Session-based authentication

---

## Tech Stack
- Backend: PHP
- Database: MySQL
- Frontend: HTML, CSS, JavaScript
